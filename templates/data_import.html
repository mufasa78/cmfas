Medicinal Materials CSV Format

Your CSV file for medicinal materials should contain the following columns:

Column Description Required
name Name of the medicinal material Yes
pinyin Pinyin transliteration No
english_name English name of the material No
province_origin Province where the material originates No
property Five properties (cold, hot, warm, cool, neutral) No
flavor Five flavors (sour, bitter, sweet, spicy, salty) No
meridian Meridian tropism (which organ/meridian it affects) No
description Description of the material No
Example:
name,pinyin,english_name,province_origin,property,flavor,meridian,description
人参,Ren Shen,Ginseng,Jilin,warm,sweet,Lung,Tonifies Qi and promotes blood production
Prescriptions CSV Format

Your CSV file for prescriptions should contain the following columns:

Column Description Required
name Name of the prescription Yes
description Description of the prescription No
efficacy What the prescription treats Yes
materials List of materials in format: "name1:amount1,name2:amount2" Yes
efficacy_categories Comma-separated list of efficacy categories No
Example:
name,description,efficacy,materials,efficacy_categories
四物汤,Classical women's tonic,Blood deficiency,当归:9g,白芍:9g,川芎:6g,熟地黄:9g,Blood tonifying,Women's health
Note: Materials must already exist in the database before importing prescriptions that reference them.
